홍길동의 웹 개발 포트폴리오

안녕하세요! 저는 웹 개발자 홍길동입니다. 최신 기술을 사용해 사용자 친화적인 웹사이트를 만듭니다.

탐색 메뉴

  • 소개
  • 프로젝트
  • 연락처

소개

저는 풀스택 웹 개발자로, 프론트엔드와 백엔드 기술을 모두 다룹니다. HTML, CSS, JavaScript뿐만 아니라 React와 Node.js도 능숙하게 사용합니다. 간결하고 효율적인 코드 작성과 사용자의 경험을 최적화하는 것을 목표로 하고 있습니다.

프로젝트

최근에 진행한 프로젝트 몇 가지를 소개합니다.

  • 온라인 쇼핑몰 개발: React와 Node.js를 사용하여 사용자 친화적인 쇼핑몰을 구축.
  • 블로그 플랫폼: 사용자 인증 및 포스트 작성 기능이 포함된 블로그 플랫폼을 개발.
  • 프로젝트 관리 도구: 팀 협업을 위한 효율적인 프로젝트 관리 웹 앱 개발.
